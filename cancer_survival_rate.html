Data Cleaning¶

  • Remove the demographic_id column. It has high cardinality and it is irrelevant.
  • Remove the year_of_death column. It has missing values for some patients that died.
  • Convert the data type of the numeric columns from object to float.
  • Encode the vital_status and paper_clinical_stage columns.

  • Fill missing values of alive patients in days_to_death column with a high value to signify that they are still alive.

  • Apply PCA to reduce the dimension of the gene expression columns.

Training with the Random Forest Algorithm¶

I use GridSearchCV hyperparameter-tuning

Since the dataset is really small, I will use 5-fold Cross-Validation for evaluation
